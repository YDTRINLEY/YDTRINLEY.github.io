Machin3tools使用笔记【Blender插件】

发表于2021-05-29|更新于2021-05-29|BlenderBlender学习笔记
|字数总计:5.7k|阅读时长:17分钟|阅读量:

视频教程链接：《还得靠它-blender2.9增强工具machin3tools 0.8.0(全讲解+辣椒中文分支+效率革命)》

¶P 常用快捷键

  • 键盘上方一行数字123，代表点线面模式
  • Ctrl 右边小键盘+：扩展选择面、点

¶P3 TAB模式饼菜单

  • 直接进入任何模式“绘制”“权重”“雕刻”“物体”“编辑”等
    雕刻模式

¶P4 5个智能顶点工具

¶1. 合并到末选点 Ctrl 1

  • 方法1：选取点，按M-合并到末选点
  • 方法2：M3工具，Ctrl 1

¶2. 合并到中点 Shift 1

¶3. 合并路径： Alt 1

  • 选中对应顶点 -> Alt 1

  • 对于顶点不相连的情况：

    • 方法1：选中边的顶点 -> 吸附工具 -> G移动吸附
    • 方法2：选中边的顶点 -> 开启右上角“自动合并顶点” -> G移动吸附
    • 方法3：Ctrl E 桥接循环边 -> 再选中一边 按X删除-有限融并
    • 方法4：直接选中4个顶点 -> Alt 1

  • 注意：如果合并路径不是所想的那样合并，可以通过Alt 1切换“拓扑”“长度”合并模式

¶4. 连接路径：Ctrl Alt 1

可以在当前有面的情况下，直接实现点对点连接成面。要注意顶点选取顺序

  • 连续填充：选择一条边，然后按住F
    在破面在面中间的情况下，直接使用”桥接循环边”会报错
  • Ctrl F栅格填充

¶5. 滑动延伸：Shift Alt 1

  • 方法1：用M3工具。最后选取的点，是滑动顶点。这种可以选取对角的点，进行滑动延伸

  • 方法2：（老方法）上方选择“活动元素”

    • 但是这个方法比较麻烦，最后还需要把质心点切换回来，但是M3工具不需要这样的切换

  • PS： GG 和 Shift A可以让顶点在边线上滑动。

    智能边(Smart Edge)
    用快捷键2+控制类按键的组合，将常见边操作进行整合，并一键化。

¶P5 智能边(Smart Edge)

¶智能边

智能边： Ctrl+2

  • 点模式-选择点数小于2时，启动切刀。
    四边面连接顶点路径，
    多边面顶点连接至末选点

  • 边模式-选非相连边旋转边 ，相连闭合边转换为面

  • 面模式-通过选择轮廓边 Ctrl+2 选择面；通过选择面，Ctrl+2 选择轮廓边

  • 投影切割(new!)-当选择一个独立的松散块，可直接在网格内容使用投影切割，注意视角。

  • 韩式倒角(new!)-倒角模式下使用,直接产生韩式倒角Korean Bevel

¶切换锐边(Toggle Sharp)

Shift+2：标记所选为锐边；重复shift 2 则为“取消标记锐边”

¶偏移边(Offset Edges)

  • 方法1：Ctrl shift r边线偏移
  • 方法2：M3工具，Alt+2偏移边线并滑移

¶P6 智能面(Smart Face)

智能面 Ctrl+3

  • 点模式：
    • 闭合顶点路径-创建多边面封闭孔洞
    • 非闭合路径 <=> 相当于F 拐点-自动封闭孔洞
      边模式下，直接双击选中轮廓边，（点模式也可以）再Ctrl 3可以直接填充面
      这种填充，甚至是根据曲率自动计算的
    • PS：双击一条边，可以选中循环边轮廓边
    • 也可以用 F2插件（详细见教程）
  • 边模式：闭合边线路径-创建多边面，进行填充传递比F填充面的效果先进； 非闭合-相当于F
    此时用Ctrl 3不会变成三角面
  • 面模式：分离所选面

¶P7 清理(Clean Up)

¶Ctrl+4:清理或选择一些模型问题

  • 移除重叠面、以角度简化模型（按角度移除面）、移除松散块孤立点边面、移除冗余、
  • 重新计算法线，翻转法线
  • 选择（可选聚焦）非流形、三角面、多边面、非平面(new!)
  • 简化：按距离合并 -> 可以减少面数，创建低面体模型减少多余并排线

¶P8 游标和原点饼菜单(Cursor and Origin Pie)

  • Shift+S：精简并添加编辑模式一键设置游标到原点、原点到游标，原点到面等更快捷工具，无需来回切模式
  • 同时在处理逻辑上增加了对游标原点旋转信息支持！！！强烈推荐
    • 对齐游标到顶点边/面
    • ALT仅设置游标位置
    • CTRL：仅设置游标旋转
  • PS：Shift 右键可以移动游标，此时会对齐我们的视角，变成另一个旋转信息。原生系统没有利用到游标的旋转（很复杂才能实现游标和旋转的双对齐）
  • 原点到游标：当选择多个物体，会根据多物体计算出中心点
  • 其他没有显示出来的是：游标->栅格；游标->活动项
  • 选中多个物体以指定物体原点为中心绕转：把原点轴心活动项物体最后选中，再原点-> 到活动

    此时要让物体的原点回到本来物体，只要选中物体原点-几何中心就好了
  • 选择合适的边线或者面，进行游标原点的旋转矫正，选中“本地/局部坐标”：让物体沿着正确的拓扑方向移动
  • 如果要开启M3工具的，“游标->栅格”，可以到插件设置开启

¶P9 对齐(Align)

¶Alt+A:快捷的对齐工具，超级好用

（new!）支持对活动物体基于两个物体对齐；

¶物体模式(Object Mode)

  • Alt+A:将选中所有物体对齐到
  • 原点：世界原点；对某些轴取消勾选，可以对某些轴取消归零
  • 游标：游标位置，相当于所选到游标
  • 活动项：对齐到活动物体，可分别决定位置旋转缩放；也可以缩放对齐
  • 基面：对齐地面(网格平面)
  • 骨骼对齐：
    选中骨骼-tab姿态模式-选中要对齐的活动骨架-Alt A ：
    • 居中对齐：新版活动项-根据两个物体“居中对齐”第三个物体位置和旋转

¶编辑模式(Edit Mode)

  • 轴向菜单：根据坐标轴对齐；
    • 根据物体原点、所选中心、游标对齐
      也可以用吸附的方法对齐
      对齐物体到边
      边长不一样，也可以边中点对齐
    • 注意：选中要对齐的几个物体进入“编辑模式”就可以，不要用A全选；如果选中了其他物体比如灯光Pointlight，则会报错
    • 根据所选顶点在坐标系数值大小对齐
      根据最大值/最小值，将曲面打平
    • 仅对齐所选的中心：本质是将当前所选面的临时中心，对齐到中心原点（物体的几何中心，世界原点）的位置
    • 根据物体原点：只要能控制当前物体原点的位置，所有选中的顶点就可以出现在当前原点的位置归零

      image.png
    • 根据物体所选中心：归零（打平）到系统临时计算出来的中心点上
    • 根据物体游标对齐（打平）：
  • 拉直
    • 点模式：根据顶点，实现快速直线拉直；
      打直的线，根据第一个和最后一个所选点的连线，去打直其他的点

    • 线模式：可以多选线，同时打直多条线；不用考虑选取的顺序，只会看线条的最开始和最后一个端点所成的连线
  • 视图方式对齐：切换到“正交视图”“顶视图”“侧视图”等转视角，根据观察者视角对齐（垂直为Y水平X）
    • 根据观察界面的上下左右来对齐顶点
    • 根据观察界面的垂直水平对齐物体原点、所选中心、游标
    • 根据观察界面的垂直水平对齐所选中心
    • 根据观察界面的垂直水平对齐 姿态模式(Pose Mode)
  • 对齐方式，对UV编辑模式也适用

¶P10 一键镜像工具(Mirror) Shift Alt 对应轴向

一键给选中物体添加镜像修改器,允许设置对称中心；直接通过这个小面板来控制“镜像修改器”，比如“切分”“翻转”“通过游标镜像”；还可以同时设置多个物体

  • X轴(X Axis)
    Shift+Alt+X:添加轴向为X的镜像修改器
  • Y轴(Y Axis)
    Shift+Alt+Y:添加轴向为Y的镜像修改器
  • Z轴(Z Axis)
    Shift+Alt+Z:添加轴向为Z的镜像修改器
  • 如果要取消批量的镜像物体，只要物体模式->选中物体->右键-M3工具-取消镜像

¶P11 视图裁剪与聚焦工具

¶裁切切换(Clipping Toggle)

  • Ctrl+下翻页（在按键End Home那一组按键里面）：一键切换视图裁切起点（非相机）
  • 在原生模式，是改“视图”菜单下的”裁剪起点

¶聚焦焦点(Focus)：独立显示

  • 聚焦所选物体，提供更易于观察SOLO分层
    原生的独立显示是按/键
  • 查看所选(View Selected)
    F：直接把视角聚焦到所选，可结合摄像机锁定使用
    忽略镜像：如果物体有镜像修改器，关闭忽略时物体的镜像部分也进行考虑
  • 局部视图(Local View)
    Ctrl+F:分层级仅显示所选物体
  • 局部视图回退(Local View invert)
    Ctrl+Alt+F:回退分层级仅显示所选物体

¶P12 文件浏览器(Filebrowser)增强工具

通过Blender文件浏览器，打开系统资源管理器

  • 开启文件浏览器(Open Filebrowser)
    O：直接在系统资源管理器打开当前文件夹
  • 排序切换(Toggle Sorting)
    切换文件排序方式
  • 缩略切换(Toggle Display)
    切换缩略图显示
  • 隐藏切换(Toggle Hidden)
    切换是否选择隐藏文件

¶P13 增强保存饼菜单(Save Pie)

  • Ctrl+S-增强保存模式，让保存不仅仅是保存文件。
  • 清理：按Alt连续清理5次
  • 打开一个文件后，会出现“上一个/下一个”，可以按顺序打开blender文件
  • - 载入常用的`世界空间源.blender`： 要载入世界环境的时候，线blender新建一个空常规文件，然后再在里面打开插件-M3工具-`世界空间源.blender`才能是绝对路径（如果不新建空文件，则载入的是相对路径） - 载入常用的`材质源.blender`：方法同上，要新建常规文件……载入后，点击材质球加载按钮

¶P14 着色和覆盖综合工具

  • 着色饼菜单(Shading Pie)上翻页(Page Up)-集成常用功能，无需再次进行翻找，随点随用，不同模式菜单显示不同（推荐设鼠标上侧键为上翻页(Page Up)）
    （new！）新增物体-按组着色
  • 渲染模式下：
    • Evee渲染器“极致”预设；辉光
  • 功能：物体命名；Alt同时开启名称；显示模式（比如“线框”）；显示轴向；法向自动光滑；光滑着色；

¶P15 视图相机增强功能

  • 视图饼菜单(View Pie)下翻页(Page Down)：
    • “转盘”和“轨迹球”的旋转方式
    • “重置视角”：让视角不倾斜，重置成朝向Y轴正方向的视角
  • 快速设置相机与视角（推荐设鼠标上侧键为上翻页(Page Down)）
  • 调节“镜头”数值，相当于直接调节“视图-焦距”

¶P16 右键菜单-右键上下文菜单(Object Context Menu)

  • 取消镜像
    移除镜像修改器
  • 选择中心对象
  • 当需要父子集一起“应用变换”的时候，用M3的应用变换 ：此时物体不会产生（不想要的）变形
  • “网格切割”：
    • 对于有“修改器”的物体，要先应用修改器，再进行“网格切割”，否则会有形变：按Alt，再按“网格切割”
    • 切割后同时“标记缝合边”：按Shift

¶P17 智能驱动和Unity导出工具

¶智能驱动(Smart Drive)

  • 快速用驱动器方式绑定控制体与被控制体，快速在两个物体之间建立连线：侧边栏“M3”
    • 比如说，控制方体在X轴方向移动的时候，球体在z方向移动：先设置“智能驱动”面板里的数值-> 选择要驱动的物体 -> 点击"驱动它"

¶Unity工具(Unity)

输出适合Unity使用的资产

¶P18 其他小功能

¶变换饼菜单Shift+上翻页：聚合变换相关设置

- 原生`,`：对应坐标系 - 原生`.`：对应质心点

¶捕捉饼菜单(Snapping Pie)

Shift+下翻页-聚合吸附相关设置

  • Shift 上翻页 ：吸附相关面板

¶集合饼菜单(Collection Pie)

Shift+C -整合集合菜单、一键选择所有集合物体

  • Shift C ：空选的时候，弹出当前所有集合菜单
    • “清除”：清除空集合
    • 选择集合对象：
      Shift 选择所有集合对象
      ALT 取消选择集合对象
      SHFT+ALT 取消选择所有集合对象
      CTRL 切换集合对象的视口选择
  • “表面滑动”：编辑模式- Tab 开启“表面滑动”。会沿着物体表面滑动点，不会离开物体表面（比GG高级，不一定沿着边线）；；再按Tab，“退出表面滑动”

¶材质选择器

  • “材质选择器”出现在界面Layout以及着色器模式：
    可以在M3工具插件界面中添加；当选中一个物体材质后，材质界面以及着色器界面，都会直接跳到该材质上，可以快速选中材质

¶工具饼菜单(Tools Pie)

替代了原生Q收藏夹-更好的兼容Boxcutter/Hardops

¶P20 M3组管理（0.6版本）类似maya/C4D

右键打开"组"界面

¶创建组

  • 物体模式-选中物体/组，Ctrl+G即可打组：
  • 面板可以改变“空物体 组”的位置

¶选择组

  • 物体模式-按住Shift+双击 物体即可根据一个物体选择整个组。（注意物体需要为非选中状态！！！）
  • 或者选中物体->右键菜单-组-选择组

¶在3D视窗隐藏组空物体

开启后组空物体被隐藏，当选中时显示。禁用此功能，则常驻显示空物体

  • 点击眼睛，全局显示空物体组；或者右键-组-勾选在3D视窗隐藏组空物体

¶自动选择组

出现于大纲或者物体模式右键菜单，开启后选择组空物体自动选择组内所有元素

¶递归选择组

出现于大纲或者物体模式右键菜单，当组层级>1时，开启后选择高级组会自动向下选择低级组

¶切换组模式

鼠标移到大纲视图，按主键盘1即可在当前和组模式进行切换

¶展开大纲视图

鼠标移到大纲视图，按主键盘2，可逐次向下展开当前集合

¶折叠大纲视图

鼠标移到大纲视图，按主键盘3，可逐次向上折叠当前集合

¶切换子级

鼠标移到大纲视图，按主键盘4，切换子级显示（即切换父子级关系显示）

¶解散组

  • 先Shift 双击选中组（包含空物体组） - 右键菜单
  • 解散当前所选所在的组(不包含更高层级)
  • Alt所选组全部解散

¶组转换

  • 对于其他方式创建的空物体为父级绑定其他子物体的组合，提供1键转换成M3组。（仅选中此结构的父级空物体才会激活）

¶复制组

非关联复制当前组。

  • 【这部分我按Alt没办法点击"复制组"，只能用下面原生模式】
  • 原生模式 Alt D:关联复制，可以关联操作物体

¶添加到组

将所选添加到活动项所在组 1先选择想添加的物体 2最后选中目标组元素

可以重新计算当前空物体组的位置

¶从组中删除

将所选从组内移除（非解散组），成为一个独立物体；再点击“重新计算当前空物体组的位置”

¶仅影响组中心

  • （复杂方法）功能同原生 -> 取消勾选“自动选择组” -> 这时才能单独选中组空物体移动它
  • 位于M3变换饼菜单下翻页，勾选组中心，开启后可移动组空物体而不影响子级变换。（相当于关闭自动选择组元素开启仅影响父级）

¶按组提供着色

  • 先改变“视图着色模式”-“物体”，才可以看到该模式下 物体颜色的改变
  • 选中要变色的物体 -> 着色饼菜单上翻页 ：
  • 改变活动项颜色，然后“着色饼菜单”-“活动”，则改组内颜色变为“活动项”颜色
  • 按材质给物体着色
  • 根据集合添加物体颜色
  • 按群组给物体着色
  • “集合”和“打组”的区别优劣：
    • “集合”：没办法确定一个中心物体，对整个“集合”作为控制，虽然可以大纲视图-右键选择所有集合物体，但是不如组方便
      但是，“集合”可以让其下的物体按照各种逻辑进行归类，而“打组”不可以
    • “组”：在其下的物体，不可以按照其他逻辑归类；

¶P21 一键螺纹+单物体投影切割+材质赋予+若干小功能（V0.6）

¶添加螺纹(new!)

  • 编辑模式 -> 选择圆柱循环面（侧楞面数>4）仅仅选择侧面（否则没效果） -> 右键-M3-添加螺纹：可自动创建螺纹，支持内外创建
  • 功能回顾：
    • 中间部分卡线“环切并滑移”：
    • 倒角： Ctrl B
    • 桥接循环边 -> 也可以在内部形成螺纹结构
    • S Shift Z：出了Z轴，沿着XY轴缩放
    • Ctrl J：物体模式 -> 选中多个物体，Ctrl J合并为一个物体

    • Shift S- 游标到所选角度 ：当转动物体而游标角度矫正的时候

¶投影切割(new!)

  • 编辑模式 -> 选中投影物体(选择一个独立的松散块) -> 旋转好视角 ->Ctrl 2，可直接在网格内容使用投影切割，注意视角。
  • 原生“投影切割”：需要是物体模式下，新建出来的独立物体，再选中切割和被切割物体，进入编辑模式，网格-投影切割
  • 区别：M3工具的“投影切割”，直接编辑模式下建立独立的切割物体就可以了
